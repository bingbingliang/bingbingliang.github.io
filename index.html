发表于 2019-03-12 | 分类于 MySql

如何使用命令行把 sql 文件导入 MySql 数据库

1.进入 MySql ：mysql -uroot -p；
（配置全局变量的情况下可直接进入，否则需要进入mysql的安装目录bin下进行进入）

2.使用需要导入到的数据库：use <库名>

3.导入 sql 文件到当前数据库：source sql 文件路径
